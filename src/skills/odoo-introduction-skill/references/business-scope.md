# Odoo 业务范围

## 核心模块分类

### 🎯 前端业务 (Front Office)

- 销售管道管理
- 客户沟通历史
- 销售活动计划
- 销售团队绩效

- 博客系统
- 表单构建器

#### 电子商务
- 产品目录管理

## 模块依赖关系

```
基础层
├── 联系人 (res.partner)
├── 产品 (product.product)
└── 公司 (res.company)

业务层
├── CRM → 销售 → 发票 → 会计
├── 采购 → 库存 → 制造
├── 项目 → 时间表 → 发票
└── 员工 → 考勤 → 薪资

扩展层
├── 网站 → 电商 → POS
├── 帮助台 → 现场服务
└── PLM → 质量 → 维护
```

## 模块安装建议

### 小微企业 (< 10 人)
**必装模块**:
- CRM
- 销售
- 发票
- 联系人
- 产品

**可选模块**:
- 网站基础
- 采购基础

### 中小企业 (10-100 人)
**必装模块**:
- 完整 CRM + 销售
- 采购 + 库存
- 完整会计
- 项目管理
- 员工管理

**可选模块**:
- 制造基础
- 网站建设
- 帮助台

### 中大型企业 (100+ 人)
**必装模块**:
- 全部核心模块
- 完整制造 (MRP)
- 完整人力资源
- 多公司配置

**可选模块**:
- PLM
- 质量管理
- 维护管理
- 营销自动化
