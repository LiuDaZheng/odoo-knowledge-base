## 什么是 Odoo

**Odoo** 是一套开源的企业资源规划（ERP）系统，采用模块化设计，提供从 CRM、销售、采购、库存、财务到人力资源等全方位业务管理功能。

### 核心特点

- **模块化架构**：按需选择和组合模块
- **开源开放**：社区版完全开源，企业版提供额外功能
- **统一平台**：所有应用共享同一数据库和界面
- **高度可定制**：支持自定义开发和第三方集成
- **现代化 UI**：基于 OWL 框架的响应式界面

## Odoo 版本历史

### 主要版本时间线

| 版本 | 发布时间 | 关键特性 |
|------|----------|----------|
| Odoo 8.0 | 2014-09 | 首次使用"Odoo"品牌，引入 Website 构建器 |
| Odoo 9.0 | 2015-10 | 企业版发布，全新 UI 设计 |
| Odoo 10.0 | 2016-10 | 社区版免费，企业版订阅制 |
| Odoo 11.0 | 2017-10 | 改进的 Studio 自定义工具 |
| Odoo 12.0 | 2018-10 | 性能优化，改进的库存管理 |
| Odoo 13.0 | 2019-10 | 全新 Website 编辑器，改进的 POS |
| Odoo 14.0 | 2020-10 | 自动化营销，改进的会计功能 |
| Odoo 15.0 | 2021-10 | OWL 框架引入，全新 UI |
| Odoo 16.0 | 2022-10 | 性能提升，改进的制造模块 |
| Odoo 17.0 | 2023-10 | 全新设计，AI 集成，性能优化 |
| Odoo 18.0 | 2024-10 | 最新 LTS 版本，改进的电商功能 |

> **当前推荐版本**: Odoo 18.0 (LTS) 或 Odoo 17.0

详细版本历史请参阅 [版本历史](references/version-history.md)。

## 业务范围

### 核心应用模块

#### 前端业务
- **CRM** - 客户关系管理
- **销售** - 报价单、销售订单
- **网站** - 网站建设、博客、电商
- **点-of-Sale** - 实体店收银系统

#### 后端运营
- **采购** - 供应商管理、采购订单
- **库存** - 仓库管理、调拨、盘点
- **制造** - BOM、工单、MRP
- **项目** - 任务管理、时间表

#### 财务管理
- **会计** - 总账、应收应付、财务报表
- **发票** - 客户/供应商发票
- **费用** - 员工报销

#### 人力资源
- **员工** - 员工档案、组织结构
- **招聘** - 职位发布、候选人跟踪
- **考勤** - 打卡记录、工时统计
- **假期** - 假期申请、余额管理

#### 其他模块
- **帮助台** - 客户服务支持
- **现场服务** - 外勤服务管理
- **租赁** - 设备租赁管理
- **订阅** -  recurring 收入管理
- **PLM** - 产品生命周期管理
- **质量** - 质量控制、警报
- **维护** - 设备维护计划
- **Sign** - 电子签名

完整模块列表请参阅 [业务范围](references/business-scope.md)。

## 版本对比

### 社区版 (Community Edition)

**免费开源**，包含核心功能：
- ✅ CRM、销售、采购、库存
- ✅ 基础会计、发票
- ✅ 项目管理、制造基础
- ✅ 网站建设基础
- ✅ 社区支持

### 企业版 (Enterprise Edition)

**订阅制**，额外功能：
- ✅ 完整会计功能（多公司、预算）
- ✅ 高级制造（MRP、PLM、质量）
- ✅ 完整人力资源套件
- ✅ 电商高级功能
- ✅ 移动应用支持
- ✅ 官方技术支持
- ✅ 自动升级服务

详细对比请参阅 [版本对比](references/edition-comparison.md)。

## 适用场景

### 按企业规模

| 规模 | 推荐版本 | 典型模块 |
|------|----------|----------|
| **小微企业** (< 10 人) | 社区版 | CRM、销售、发票、库存基础 |
| **中小企业** (10-100 人) | 企业版 | 完整财务、采购、库存、制造 |
| **中大型企业** (100+ 人) | 企业版 + 定制 | 多公司、高级 MRP、完整 HR |

### 按行业

| 行业 | 核心模块 | 特殊需求 |
|------|----------|----------|
| **制造业** | 制造、库存、采购、质量 | BOM 管理、工单跟踪、MRP |
| **零售业** | POS、库存、销售、电商 | 多渠道销售、库存同步 |
| **服务业** | 项目、时间表、帮助台 | 工时跟踪、服务级别协议 |
| **批发贸易** | 采购、库存、销售、会计 | 多仓库、批次跟踪 |
| **电商** | 网站、电商、库存、会计 | 支付集成、订单履行 |

详细行业方案请参阅 [适用场景](references/use-cases.md)。
